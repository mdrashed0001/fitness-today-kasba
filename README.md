# Fitness Today - Kasba

A modern, responsive website for Fitness Today Gym in Kasba, Kolkata.

## Features
- ✅ Responsive design for all devices
- ✅ Photo gallery with lightbox modal
- ✅ Membership package pricing
- ✅ Contact information with Google Maps
- ✅ Mobile-friendly menu toggle
- ✅ Smooth scrolling navigation

## Live Website
📍 **Visit:** https://mdrashed0001.github.io/fitness-today-kasba/

## Files
- `index.html` - Main HTML structure
- `style.css` - Styling and responsive design
- `script.js` - JavaScript functionality
- `README.md` - Documentation

## Setup
1. Clone the repository
2. Open `index.html` in your browser
3. Update Google Maps API key in the contact section

## Important Notes
⚠️ **Google Maps API Key**: Replace `YOUR_ACTUAL_GOOGLE_MAPS_API_KEY` in index.html with your actual Google Maps API key for the map to display properly.
